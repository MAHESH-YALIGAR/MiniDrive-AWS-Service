Mini Drive is a full-stack cloud storage application built using the MERN Stack and AWS S3. It allows users to securely upload, manage, share, restore, and permanently delete files, images, videos, and folders through an intuitive web interface.

## ✨ Features


### ☁️ AWS S3 Cloud Storage

* Store files securely in AWS S3
* Upload Images
* Upload Videos
* Upload Documents (PDF, DOCX, PPT, etc.)
* Upload Folders

### 📂 File Management

* View all uploaded files in a centralized dashboard
* Search files instantly
* List View and Grid View modes
* Download files
* Delete files

### 🔗 File Sharing

* Generate shareable file links
* Copy link and share files with others

### 🗑️ Trash Management

* Move files to Trash
* Restore deleted files from Trash
* Permanently delete files


### 📁 Storage Organization

* My Files & Photos
* My Videos
* My Folders
* Shared with Me
* Recent Files
* Trash Bin

### 📊 Storage Monitoring

* Display used storage space
* Track available storage capacity
* Storage usage percentage indicator

### 🎨 User Interface

* Responsive dashboard design
* Dark mode interface
* Easy navigation sidebar
* File preview thumbnails for images

## 💻 How to Run Locally

Follow these steps to spin up both the backend and frontend components of the application on your local machine:

### 1. Clone the Repository
```bash
https://github.com/MAHESH-YALIGAR/MiniDrive-AWS-Service/tree/main/frontend
```

### 2. Run the Backend API
Open a new terminal window and navigate to the backend directory:
```bash
cd backend
# Install Node modules and dependencies
npm init i
# Start the frontend local development server
npm start
### 3. Run the Frontend Interface
Open a second terminal window and navigate to the frontend directory:
```bash
cd frontend

# Install Node modules and dependencies
npm install

# Start the frontend local development server
npm run dev
