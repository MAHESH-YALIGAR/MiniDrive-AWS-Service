Mini Drive is a full-stack cloud storage application built using the MERN Stack and AWS S3. It allows users to securely upload, manage, share, restore, and permanently delete files, images, videos, and folders through an intuitive web interface.

## ✨ Features


### ☁️ AWS S3 Cloud Storage

* Store files securely in AWS S3
* Upload Images
* Upload Videos
* Upload Documents (PDF, DOCX, PPT, etc.)
* Upload Folders

### 📂 File Management

* View all uploaded files in a centralized dashboard
* Search files instantly
* List View and Grid View modes
* Download files
* Delete files

### 🔗 File Sharing

* Generate shareable file links
* Copy link and share files with others

### 🗑️ Trash Management

* Move files to Trash
* Restore deleted files from Trash
* Permanently delete files


### 📁 Storage Organization

* My Files & Photos
* My Videos
* My Folders
* Shared with Me
* Recent Files
* Trash Bin

### 📊 Storage Monitoring

* Display used storage space
* Track available storage capacity
* Storage usage percentage indicator

### 🎨 User Interface

* Responsive dashboard design
* Dark mode interface
* Easy navigation sidebar
* File preview thumbnails for images
